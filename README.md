# MKNotification

Demo: https://wpajans.net/jQuery/MKNotification/

Properties;

- Music (Optional)
- Image (Optional)
- Timer (Optional)
- Easy Edit
- Easy USage


# Usage
```
 $(function(){
    $("body").MKNotification({
      'MKMessage' : 'Example, Message!',
      'MKDelay' : '1500',
      'MKIcon' : 'http://icons.iconarchive.com/icons/custom-icon-design/flatastic-2/128/success-icon.png',
      'MKSound' : 'refresh.mp3'
    });
});
 ```
